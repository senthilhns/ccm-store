This repo contains Harness CCM terraform scripts for the video demos of Auto Stopping and Cost Monitoring

These scripts are meant for understanding the automation process using Harness Terraform provider for Harness CCM Features. Use these scripts after understanding them thoroughly and considering the cost, security considerations carefully for your testing needs.

For some of the code examples the ports and permissions may be kept wide open only for testing to isolate cloud permission issues. These permissions need to be checked thoroughly and narrowed down as per your security requirements.
